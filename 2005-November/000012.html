Hi,

Not that I plan to release the next days, but I added a new feature
that I needed to SPE. It's a pydoc documentation browser. It's next to
UML in the tab. Please try it out and see if there are any bugs. If
you press on:
- most links: open documentation about that module
- index: show an overview of all available modules on your system
- a filename: opens this file in SPE

One thing to be careful of is that pydoc needs to import the file to
generate the documentation. Therefore SPE warns you to save files, in
case you didn't.
