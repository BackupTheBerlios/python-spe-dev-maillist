Notebook View fixed for OSX
Tested with P2.5 and wx2.8.1.
Please confirm with other versions.
Replace as_notebook function in parent.py with below:


     def as_notebook(self,event):
         if hasattr(self.frame,'Tile'):
             self.app.childActive.frame.Maximize()
         else:
             tabs  =  getattr(self.frame,'tabs',None)
             if tabs:
                 # Have to set selection to first tab and de-Tile before
                 # starting iterations to show sidebars etc
                 tabs.ShowTabs(True)
                 tabs.SetSelection(0)
                 tabs.Tile(False)
                 for i in range(tabs.GetPageCount()):
                     tabs.SetSelection(i)
                     tabs.Tile(False)
                     self.app.childActive.source.SetFocus()
                     self.app.childActive.showSidebar()
