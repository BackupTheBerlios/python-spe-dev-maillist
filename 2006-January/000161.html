Hi
I tested it on the Mac and the default font was to small. I've thought 14
would be a good option. You can play with it yourself by editing the
_spe/dialogs/stcStyleEditor.py
in StcStyleEditor.py:

def getDefs():
    if wx.Platform == '__WXMSW__':
        commonDefs = {
            'helv': 'Courier New',
            'mono': 'Courier New',
            'lnsize': 10,
            'backcol': '#FFFFFF',
            'size': 10}
    elif wx.Platform =='__WXMAC__':
        commonDefs = {
            'helv': 'Helvetica',
            'mono': 'Courier',
            'lnsize': 14,
            'backcol': '#FFFFFF',
            'size': 14}
    else:
        commonDefs = {
            'helv': 'Helvetica',
            'mono': 'Courier',
            'lnsize': 10,
            'backcol': '#FFFFFF',
            'size': 10}

You can configure the colors by pressing the button
'Preferences>Editor>Configure Style...'. You can choose between Boa, Idle,
SPE and other styles.

Mac issues:
The dialog was taken from the sample in the wxPython directory and
unfortunately doesn't display well on Mac, but luckily good enough to be
minimal operational by using the speedsetting. Another problem on Mac is
that I had to change the style chatting twice to idle in order it to work.
I'll look into that wednesday.
